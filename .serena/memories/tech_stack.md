## フロントエンド
- **Next.js 14** (App Router)
- **TypeScript**
- **React 18**
- **Tailwind CSS**
- **React Query (TanStack Query)**

## バックエンド

- **Docker & Docker Compose**
- **Docker MCP CLI**
- **Bitwarden CLI**

- **Playwright** (E2E テスト)

- **Prettier**: コードフォーマット
